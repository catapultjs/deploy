# `version`

Prints the current `@catapultjs/deploy` version.

## Usage

```bash
npx cata version [options]
```

## Options

| Option | Description |
| --- | --- |
| `--json` | Output result as JSON |

## Examples

```bash
npx cata version
npx cata version --json
```

This command does not require a deploy config file.
